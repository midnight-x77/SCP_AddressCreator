# Pi Testnet 지갑 생성기 (니모닉 기반)

이 프로젝트는 Pi Testnet에서 니모닉(비밀 구절)을 기반으로 새로운 지갑을 생성하고, 초기 잔액으로 1 Pi를 충전하는 스크립트입니다.

## 기능

- **24단어 니모닉 생성**: 안전한 256비트 엔트로피를 기반으로 비밀 구절을 생성합니다.
- **표준 키 파생**: BIP39 시드와 스텔라/파이 표준 파생 경로(`m/44'/148'/0'`)를 사용하여 키를 도출합니다.
- **계정 자동 활성화**: 생성자 지갑(Creator Wallet)으로부터 1 Pi를 전송하여 새로운 지갑 계정을 생성 및 활성화합니다.

## 사전 준비

- [Node.js](https://nodejs.org/) (v14 이상 권장)가 설치되어 있어야 합니다.
- Pi Testnet에서 잔액이 있는 **생성자 지갑의 시크릿 키(Secret Key)**가 필요합니다.

## 설치 방법

1. 저장소를 클론하거나 프로젝트 폴더로 이동합니다.
2. 필요한 패키지를 설치합니다:

```bash
npm install
```

## 사용 방법

스크립트 실행 시 생성자 지갑의 시크릿 키를 `-s` 파라미터로 전달해야 합니다.

### 테스트넷 (기본값)
```bash
node create-wallet.js -s <SECRET_KEY>
```

### 메인넷
```bash
node create-wallet.js -mainnet -s <SECRET_KEY>
```

## 파라미터 설명

- `-s <SECRET_KEY>`: (필수) 잔액이 있는 생성자 지갑의 시크릿 키입니다.
- `-mainnet`: (선택) 지정 시 Pi 메인넷 네트워크를 사용합니다. 지정하지 않으면 테스트넷이 기본값입니다.

## 키 파생 과정 설명

본 스크립트는 다음의 표준 절차를 따릅니다:

1. **비밀 구절 생성**: 24개의 영단어로 구성된 니모닉을 생성합니다.
2. **마스터 시드 생성**: 니모닉으로부터 512비트 바이너리 시드를 도출합니다.
3. **파생 경로 적용**: `m/44'/148'/0'` 경로를 적용하여 Ed25519 키를 도출합니다.
4. **키 쌍 도출**: 도출된 키로부터 스텔라/파이 호환 공개키(G...)와 시크릿키(S...)를 생성합니다.

## 주의사항

- **비밀 구절 및 시크릿 키 보안**: 출력된 비밀 구절과 시크릿 키는 절대 타인에게 노출해서는 안 됩니다.
- **네트워크 설정**: 이 코드는 기본적으로 Pi Testnet(`https://api.testnet.minepi.com`)을 바라보고 있습니다.

## 라이브러리 정보

- `@stellar/stellar-sdk`: 스텔라/파이 네트워크 통신 및 트랜잭션 빌드
- `bip39`: 니모닉 생성 및 시드 도출
- `ed25519-hd-key`: Ed25519 기반 계층적 결정론적(HD) 키 파생
