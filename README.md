# SCP 주소 생성기 (니모닉 기반)

이 프로젝트는 SCP 블록체인(Stellar/Pi Network) 생태계에서 Stellar-SDK를 이용해 니모닉(비밀 구절) 기반의 새로운 지갑 주소를 생성하는 스크립트입니다. 테스트넷(Testnet)과 메인넷(Mainnet) 네트워크를 모두 지원하며, 24단어 비밀 구절에서 표준 파생 경로를 통해 키 쌍을 도출합니다. 소스코드에는 Pi Network의 RPC URL이 등록되어 있으며, Stellar RPC URL로 교체하면 Stella에서도 동작합니다.

## 주요 기능

- **네트워크 완벽 지원**: 파라미터 설정을 통해 Pi Testnet과 Mainnet을 모두 원활하게 사용할 수 있습니다.
- **24단어 니모닉 생성**: 안전한 256비트 엔트로피를 기반으로 비밀 구절을 생성합니다.
- **표준 키 파생**: BIP39 시드와 스텔라/파이 표준 파생 경로(`m/44'/148'/0'`)를 사용하여 암호학적으로 안전한 키를 도출합니다.
- **계정 자동 활성화**: 생성자 지갑(Creator Wallet)으로부터 1 Pi를 전송하여 새로운 지갑 계정을 생성 및 활성화합니다.

## 사전 준비

- [Node.js](https://nodejs.org/) (v14 이상 권장) 설치 환경
- Pi 네트워크(Testnet 또는 Mainnet)에 잔액이 존재하는 **생성자 지갑의 시크릿 키(Secret Key)**

## 설치 방법

1. 저장소를 클론하거나 프로젝트 폴더로 이동합니다.
2. 터미널에서 필요한 패키지를 설치합니다:

```bash
npm install
```

## 사용 방법

명령어 실행 시 생성자 지갑의 시크릿 키를 `-s` 파라미터로 전달해야 합니다. 기본적으로 코드는 테스트넷 노드에 연결됩니다.

### 테스트넷 (기본값)
```bash
# SECRET_KEY는 S로 시작하는 비밀키이며, Pi Wallet의 지갑설정에서 확인할 수 있습니다.
# <SECRET_KEY>의 '<'와 '>'는 입력이 아님
node create-wallet.js -s <SECRET_KEY>

node create-wallet.js -s SBQPD**********BYRCUOPR2M3FHP365ZXGVDSPHHQT6VQ6MZ3ZV4SWS
```
```bash
# 결과출력
네트워크: Testnet
생성자 주소: 

=== 새로운 지갑 키페어 생성 완료 ===
비밀 구절(Passphrase): 
공개키(Public Key): 
시크릿키(Secret Key): 
====================================

생성자 계정 정보를 파이 테스트넷에서 불러오는 중...
트랜잭션 생성 중... (초기 잔액 1 Pi 지급)
트랜잭션을 네트워크에 제출하는 중...

성공! 트랜잭션 해시: 
새로운 지갑이 파이 테스트넷에 성공적으로 생성되었으며 1 Pi가 충전되었습니다.
```

### 메인넷
메인넷 환경에서 주소를 생성하고 활성화하려면 `-mainnet` 플래그를 추가하십시오.
```bash
node create-wallet.js -mainnet -s <SECRET_KEY>
```

## 파라미터 상세

- `-s <SECRET_KEY>`: (필수) 계정 활성화를 위한 생성자 지갑의 시크릿 키입니다. 반드시 지갑활성화를 시킬 1토큰과 수수료가 있어야 합니다.
- `-mainnet`: (선택) 이 옵션을 포함하면 Mainnet 네트워크로 통신합니다. 생략할 경우 Testnet을 기본으로 사용합니다.

## 키 파생 과정 (Technical Details)

본 스크립트는 암호화폐 표준 절차를 철저히 준수합니다:

1. **비밀 구절 생성**: 24개의 영단어로 구성된 니모닉을 무작위로 생성합니다.
2. **마스터 시드 생성**: 니모닉으로부터 512비트의 바이너리 시드를 도출합니다.
3. **파생 경로 적용**: 스텔라 합의 프로토콜 기반의 `m/44'/148'/0'` 경로를 거쳐 Ed25519 기반의 키를 안전하게 도출합니다.
4. **키 쌍 도출**: 완성된 키에서 스텔라/파이 호환의 공개키(G...)와 시크릿키(S...)를 생성합니다.

## 보안 주의사항

- **비밀 구절 및 시크릿 키의 안전성**: 화면에 출력된 24단어 비밀 구절과 시크릿 키(S...)는 지갑의 통제권을 의미합니다. 절대 인터넷 공간이나 타인에게 노출하지 마시고 오프라인 환경에서 안전하게 보관하십시오.

## 사용된 오픈소스 라이브러리

- `@stellar/stellar-sdk`: 네트워크와의 통신 및 트랜잭션 빌드를 위한 스텔라 SDK
- `bip39`: 표준 니모닉 생성 및 시드 도출 알고리즘
- `ed25519-hd-key`: Ed25519 타원곡선 기반 계층적 결정론적(HD) 키 파생
